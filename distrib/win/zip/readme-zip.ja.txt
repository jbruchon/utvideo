# Ut Video Codec Suite  ZIP パッケージ  readme


## 警告

readme.ja.html と gplv2.txt に記載されたライセンス条項に同意した場合にのみ Ut Video Codec Suite のインストールと使用ができます。

Ut Video Codec Suite をインストール/アンインストールする前に、全てのアプリケーションを終了するべきです(SHOULD)。

このバージョンをインストールする前に、古いバージョンをアンインストールするべきです(SHOULD)。

将来のバージョンをインストールする前にも、このバージョンをアンインストールするべきです(SHOULD)。


## インストール

1. "install.bat" を右クリックし、コンテキストメニューの "管理者として実行" をクリックします。
   - ユーザーアカウント制御ダイアログが表示された場合は "はい" をクリックします。
2. ライセンスに同意するかどうかの確認が（英語で）表示されるので、その内容に従います。
3. インストールが完了し、Webブラウザでいくつかのドキュメントが開きます。


## アンインストール

1. "uninstall.bat" を右クリックし、コンテキストメニューの "管理者として実行" をクリックします。
   - ユーザーアカウント制御ダイアログが表示された場合は "はい" をクリックします。
